Formalism

Let us now abstract to the simplest scenario where Alice has only two possible public keys, and Bob has only two possible messages to communicate with Alice. Realize that the fundamental logic of this abstraction holds symmetrically for communication in the opposite direction, as well as iteravily for any number of possible keys and messages.
